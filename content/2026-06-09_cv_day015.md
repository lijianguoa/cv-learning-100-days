## 📜 历史背景

模板匹配是计算机视觉中最古老、最直观的模式识别方法之一。其核心思想可以追溯到20世纪60年代的模式识别研究：

- **1966年**，Hall提出了一种基于互相关的图像匹配方法，奠定了模板匹配的理论基础。
- **1971年**，Anuta系统性地研究了数字图像配准中的快速互相关技术，大幅提升了计算效率。
- **1987年**，Lewis在论文"Fast Template Matching"中提出了利用积分图像加速归一化互相关（NCC）计算的方法。
- **2006年**，OpenCV正式集成了`cv2.matchTemplate()`函数，使模板匹配成为最易上手的CV工具之一。

模板匹配的发展与**信号处理**和**统计模式识别**密切相关。其核心——互相关运算，本质上源自信号处理中的**互相关函数**（Cross-correlation function），用于衡量两个信号在不同时间偏移下的相似度。

## 🧠 核心概念

### 1. 模板匹配基本原理

模板匹配是一种**滑动窗口**搜索技术：将一个小的模板图像（template）在输入图像（source image）上逐像素滑动，在每个位置计算相似度得分，最终找到得分最高（或最低）的位置作为匹配结果。

```
得分图 R(x,y) = Σ Σ T(x',y') · I(x+x', y+y')
```

其中，`T` 为模板图像，`I` 为输入图像，`(x,y)` 为当前位置。

### 2. OpenCV支持的六种匹配方法

| 方法名称 | 公式 | 特点 |
|---------|------|------|
| `TM_SQDIFF` | Σ(T-I)² | 平方差匹配，值越小越匹配 |
| `TM_SQDIFF_NORMED` | 归一化平方差 | 消除亮度影响 |
| `TM_CCORR` | Σ(T·I) | 互相关匹配，值越大越匹配 |
| `TM_CCORR_NORMED` | 归一化互相关 | 值域[0,1]，更鲁棒 |
| `TM_CCOEFF` | Σ(T'·I') | 相关系数匹配，消除均值偏移 |
| `TM_CCOEFF_NORMED` | 归一化相关系数 | **最推荐**，值域[-1,1] |

> 💡 **最佳实践**：推荐使用 `TM_CCOEFF_NORMED`（归一化相关系数法），它对光照变化和对比度差异具有最好的鲁棒性。

### 3. 归一化互相关（NCC）

归一化互相关是最常用的模板匹配度量：

$$
R(x,y) = \frac{\sum_{x',y'} (T(x',y') - \bar{T}) \cdot (I(x+x',y+y') - \bar{I}_{x,y})}{\sqrt{\sum_{x',y'} (T(x',y') - \bar{T})^2 \cdot \sum_{x',y'} (I(x+x',y+y') - \bar{I}_{x,y})^2}}
$$

其中 $\bar{T}$ 是模板均值，$\bar{I}_{x,y}$ 是图像在位置(x,y)处的局部均值。

### 4. 多尺度模板匹配

当目标在图像中的大小与模板不一致时，单尺度匹配会失效。多尺度方法通过**图像金字塔**实现。

### 5. 模板匹配的局限性

- ❌ **旋转不变性差**：目标旋转后匹配失败
- ❌ **尺度敏感**：目标大小变化时需要多尺度处理
- ❌ **遮挡脆弱**：目标被部分遮挡时效果差
- ❌ **非刚性形变**：无法处理形变目标
- ✅ 适合：**刚性物体、固定视角、可控光照**的场景

## 🧪 经典实验

### 实验1：基础模板匹配

```python
result = cv2.matchTemplate(img, template, cv2.TM_CCOEFF_NORMED)
min_val, max_val, min_loc, max_loc = cv2.minMaxLoc(result)
top_left = max_loc
bottom_right = (top_left[0] + w, top_left[1] + h)
cv2.rectangle(img, top_left, bottom_right, 255, 2)
```

### 实验2：多目标模板匹配

```python
threshold = 0.8
loc = np.where(result >= threshold)
for pt in zip(*loc[::-1]):
    cv2.rectangle(img, pt, (pt[0] + w, pt[1] + h), (0, 255, 0), 2)
```

### 实验3：多尺度模板匹配实战

```python
for scale in np.linspace(0.3, 1.5, 30):
    resized = cv2.resize(gray, None, fx=scale, fy=scale)
    result = cv2.matchTemplate(resized, template, cv2.TM_CCOEFF_NORMED)
    _, max_val, _, max_loc = cv2.minMaxLoc(result)
```
