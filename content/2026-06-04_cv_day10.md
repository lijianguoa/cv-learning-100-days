## 📜 历史背景

图像金字塔的概念起源于**信号处理中的多分辨率分析**（Multiresolution Analysis）。1983年，**Peter J. Burt** 和 **Edward H. Adelson** 在经典论文 *"A Multiresolution Spline With Application to Image Mosaics"* 中首次系统性地提出了**拉普拉斯金字塔**（Laplacian Pyramid），并将其应用于图像融合与拼接领域。

在此之前，**高斯金字塔**的思想已经隐含在**高斯核的多尺度采样**中。1980年，**Stefano Alliney** 和 **Carlo Morandi** 等人在数字图像处理研究中，探索了图像的多尺度表示方法。

图像金字塔的核心思想是：**通过对图像进行逐层降采样和插值重建，构建一个从粗到细的多分辨率表示体系**。这一思想后来成为**SIFT**、**图像拼接**、**目标检测多尺度策略**等众多经典算法的基础。

## 🧠 核心概念

### 1. 图像金字塔概述

图像金字塔是一种**多尺度图像表示方法**，通过逐层对图像进行**高斯模糊**和**降采样**，生成一系列分辨率递减的图像，形似金字塔结构。

```
        ┌─────┐
       ┌┤  G0 ├┐          ← 原始图像 (最高分辨率)
      ┌┤ └─────┘ ├┐
     ┌┤  ┌─────┐  ├┐       ← 第1层降采样 (1/2分辨率)
    ┌┤  │  G1  │  ├┐
   ┌┤   └─────┘   ├┐       ← 第2层降采样 (1/4分辨率)
  ┌┤    ┌─────┐    ├┐
 ┌┤     │  G2  │     ├┐    ← 第3层降采样 (1/8分辨率)
 └┤      └─────┘      ├┘
  └───────────────────┘
```

### 2. 高斯金字塔 (Gaussian Pyramid)

高斯金字塔通过以下步骤构建：

1. **高斯模糊**：对当前层图像进行高斯卷积（核大小通常为5×5）
2. **降采样**：将模糊后的图像隔行隔列采样，尺寸缩小为原来的1/4（宽高各1/2）

**数学表达**：

$$G_0 = \text{原始图像}$$

$$G_{l}(i,j) = \sum_{m=-2}^{2}\sum_{n=-2}^{2} w(m,n) \cdot G_{l-1}(2i+m, 2j+n)$$

其中 $w(m,n)$ 为5×5高斯核权重：

$$w = \frac{1}{256} \begin{bmatrix} 1 & 4 & 6 & 4 & 1 \\ 4 & 16 & 24 & 16 & 4 \\ 6 & 24 & 36 & 24 & 6 \\ 4 & 16 & 24 & 16 & 4 \\ 1 & 4 & 6 & 4 & 1 \end{bmatrix}$$

**OpenCV实现**：

```python
import cv2

img = cv2.imread('input.jpg')

# 向下采样（降采样）
lower_reso = cv2.pyrDown(img)
lower_reso2 = cv2.pyrDown(lower_reso)

# 向上采样（升采样）
higher_reso = cv2.pyrUp(lower_reso)
```

> 📖 **参考文档**：[OpenCV pyrDown 官方文档](https://docs.opencv.org/4.x/d4/d1f/tutorial_pyramids.html)

### 3. 拉普拉斯金字塔 (Laplacian Pyramid)

拉普拉斯金字塔存储的是**高斯金字塔相邻两层之间的差异信息**，可以理解为图像的**高频细节**。

**构建方法**：

$$L_l = G_l - \text{expand}(G_{l+1})$$

其中 $\text{expand}(G_{l+1})$ 是将 $G_{l+1}$ 上采样到与 $G_l$ 相同的尺寸。

**物理意义**：
- 拉普拉斯金字塔的每一层捕获了该尺度下的**边缘和纹理细节**
- 底层包含**高频细节**（边缘、纹理）
- 顶层包含**低频信息**（整体轮廓）

**OpenCV实现**：

```python
import cv2
import numpy as np

img = cv2.imread('input.jpg')

# 构建高斯金字塔
G = [img.copy()]
for i in range(4):
    G.append(cv2.pyrDown(G[-1]))

# 构建拉普拉斯金字塔
L = []
for i in range(len(G)-1):
    GE = cv2.pyrUp(G[i+1], dstsize=(G[i].shape[1], G[i].shape[0]))
    L.append(cv2.subtract(G[i], GE))

# 从拉普拉斯金字塔重建图像
reconstructed = L[0].copy()
for i in range(1, len(L)):
    reconstructed = cv2.add(reconstructed, cv2.pyrUp(G[i], dstsize=(reconstructed.shape[1], reconstructed.shape[0])))
```

### 4. 图像融合 (Image Blending)

利用拉普拉斯金字塔可以实现**无缝图像融合**：

1. 分别构建两幅图像的高斯金字塔和拉普拉斯金字塔
2. 为每层创建一个**掩码金字塔**（Gaussian Pyramid of Mask）
3. 在拉普拉斯金字塔的每一层，使用掩码进行加权融合
4. 从顶层向下逐层重建，得到最终融合结果

```python
def pyramid_blend(imgA, imgB, mask):
    # 构建A和B的高斯金字塔
    gpA = [imgA.copy()]
    gpB = [imgB.copy()]
    gpMask = [mask.copy()]

    for i in range(6):
        gpA.append(cv2.pyrDown(gpA[-1]))
        gpB.append(cv2.pyrDown(gpB[-1]))
        gpMask.append(cv2.pyrDown(gpMask[-1]))

    # 构建拉普拉斯金字塔
    lpA = [gpA[-1]]
    lpB = [gpB[-1]]
    gpMask_reversed = gpMask[::-1]

    for i in range(5, 0, -1):
        size = (gpA[i-1].shape[1], gpA[i-1].shape[0])
        GE_A = cv2.pyrUp(gpA[i], dstsize=size)
        GE_B = cv2.pyrUp(gpB[i], dstsize=size)
        lpA.append(cv2.subtract(gpA[i-1], GE_A))
        lpB.append(cv2.subtract(gpB[i-1], GE_B))

    # 逐层融合
    blended = []
    for la, lb, gm in zip(lpA, lpB, gpMask_reversed):
        rows, cols, dpt = la.shape
        gm_resized = cv2.resize(gm, (cols, rows))
        gm_3ch = cv2.merge([gm_resized]*3) / 255.0
        blended.append(cv2.add(np.float32(la) * gm_3ch,
                              np.float32(lb) * (1 - gm_3ch)))

    # 重建
    result = blended[-1]
    for i in range(len(blended)-2, -1, -1):
        result = cv2.pyrUp(result, dstsize=(blended[i].shape[1], blended[i].shape[0]))
        result = cv2.add(result, blended[i])

    return result
```

## 👨‍🔬 科学家故事

### Peter J. Burt (彼得·伯特)

Peter Burt 是**罗切斯特理工学院（RIT）**的计算机科学教授，他在**多分辨率图像处理**领域做出了奠基性贡献。1983年，他与Edward Adelson共同提出的**拉普拉斯金字塔**至今仍是图像处理教科书中的经典内容。

Burt的研究兴趣广泛，涵盖**人类视觉系统建模**、**图像压缩**和**多传感器数据融合**。他提出的**Burt-Adelson金字塔**不仅在学术界影响深远，还被广泛应用于**JPEG2000图像压缩标准**和**数字电影后期制作**中。

有趣的是，Burt最初的研究动机是理解**人眼如何在不同距离下感知场景**——这正是多分辨率分析的生物学基础。

### Edward H. Adelson (爱德华·阿德尔森)

Edward Adelson 是**MIT脑与认知科学系**和**计算机科学与人工智能实验室（CSAIL）**的教授。他是一位跨越**视觉感知**和**计算机视觉**的传奇学者。

Adelson最广为人知的成就是创造了**棋盘阴影错觉**（Checker Shadow Illusion），这一视觉错觉展示了人脑对光照补偿的强大能力。他还发明了**可编程像素**（programmable pixel）技术，对计算摄影学产生了深远影响。

### 实验2：拉普拉斯金字塔重建验证

```python
import cv2
import numpy as np

img = cv2.imread('input.jpg', cv2.IMREAD_GRAYSCALE)
img = img.astype(np.float32)

# 构建高斯金字塔
G = [img.copy()]
for i in range(4):
    G.append(cv2.pyrDown(G[-1]))

# 构建拉普拉斯金字塔
L = []
for i in range(len(G)-1):
    GE = cv2.pyrUp(G[i+1], dstsize=(G[i].shape[1], G[i].shape[0]))
    L.append(G[i] - GE)

# 从拉普拉斯金字塔重建
reconstructed = G[-1].copy()
for i in range(len(L)-1, -1, -1):
    reconstructed = cv2.pyrUp(reconstructed, dstsize=(L[i].shape[1], L[i].shape[0]))
    reconstructed += L[i]

# 验证重建误差
error = np.abs(img - reconstructed)
print(f"最大重建误差: {error.max():.6f}")
print(f"平均重建误差: {error.mean():.6f}")
```

## 📖 小故事

### 金字塔与古埃及的奇妙联系

图像金字塔的命名并非偶然——它的形状确实酷似古埃及的金字塔：底层宽大（高分辨率），顶层窄小（低分辨率）。有趣的是，古埃及人建造金字塔时就已经运用了**多尺度思维**：从远处看，金字塔是一个完美的三角形；走近看，每一块石头都有精细的雕刻。

这与图像金字塔的哲学不谋而合：**远观得其形，近察得其细**。计算机视觉中的金字塔正是让算法同时具备"远观"和"近察"的能力。

### SIFT中的金字塔：一个价值数十亿的想法

2000年，David Lowe在UBC（不列颠哥伦比亚大学）提出了SIFT算法。这个算法的核心就是在图像金字塔上进行多尺度特征检测。有趣的是，SIFT最初被申请了专利，这导致它在很长一段时间内无法被 freely 使用。直到2020年3月专利到期，SIFT才终于被纳入OpenCV的主分支。

这个故事告诉我们：**好的想法可能被暂时封锁，但知识终将自由流动**。
