## 历史背景

相机标定的历史可以追溯到19世纪的摄影测量学。1839年，法国科学家[达盖尔](https://zh.wikipedia.org/wiki/路易·达盖尔)发明了银版摄影法，开启了用相机记录世界的时代。但直到20世纪中叶，人们才开始系统地研究如何从二维图像恢复三维信息。

1950年代，[摄影测量学](https://zh.wikipedia.org/wiki/摄影测量学)在测绘领域快速发展，标定技术主要用于航空摄影。1970年代，[Roger Tsai](https://en.wikipedia.org/wiki/Roger_Y._Tsai)提出了经典的Tsai标定法，首次将镜头畸变模型引入相机标定，奠定了现代相机标定的理论基础。

1987年，[Zhengyou Zhang](https://en.wikipedia.org/wiki/Zhengyou_Zhang)（张正友）在微软研究院提出了著名的[张正友标定法](https://en.wikipedia.org/wiki/Zhang%27s_camera_calibration_algorithm)，利用平面标定板进行标定，大大简化了标定流程，成为目前最广泛使用的标定方法。

进入21世纪，随着计算机视觉和机器人技术的蓬勃发展，相机标定已成为自动驾驶、三维重建、增强现实等领域的核心基础技术。

## 核心概念

### 1. 相机模型

相机成像过程可以用[针孔相机模型](https://en.wikipedia.org/wiki/Pinhole_camera_model)来描述：

$$
Z \begin{bmatrix} u \\ v \\ 1 \end{bmatrix} = K [R|t] \begin{bmatrix} X \\ Y \\ Z \\ 1 \end{bmatrix}
$$

其中：
- $K$ 为**内参矩阵**（Intrinsic Matrix）
- $[R|t]$ 为**外参矩阵**（Extrinsic Matrix）
- $(X,Y,Z)$ 为世界坐标系中的三维点
- $(u,v)$ 为图像坐标系中的二维点

**内参矩阵** $K$：
$$
K = \begin{bmatrix} f_x & 0 & c_x \\ 0 & f_y & c_y \\ 0 & 0 & 1 \end{bmatrix}
$$

其中 $f_x, f_y$ 为焦距（像素单位），$c_x, c_y$ 为主点坐标。

### 2. 镜头畸变

实际相机镜头存在多种畸变，主要包括：

| 畸变类型 | 描述 | 数学模型 |
|---------|------|---------|
| **径向畸变** | 光线在镜头边缘弯曲 | $x_{distorted} = x(1 + k_1r^2 + k_2r^4 + k_3r^6)$ |
| **切向畸变** | 镜头与成像平面不平行 | $x_{distorted} = x + [2p_1xy + p_2(r^2+2x^2)]$ |

畸变系数：$[k_1, k_2, p_1, p_2, k_3]$

### 3. 单目相机标定

[单目相机标定](https://docs.opencv.org/4.x/dc/dbb/tutorial_py_calibration.html)流程：

1. **准备标定板**：打印棋盘格或圆点标定板
2. **采集图像**：从不同角度拍摄15-20张标定板图像
3. **角点检测**：使用`cv2.findChessboardCorners()`检测角点
4. **亚像素精化**：使用`cv2.cornerSubPix()`提高精度
5. **标定计算**：使用`cv2.calibrateCamera()`计算参数
6. **精度评估**：计算重投影误差

**OpenCV标定代码示例**：
```python
import cv2
import numpy as np

# 准备对象点
objp = np.zeros((6*9, 3), np.float32)
objp[:, :2] = np.mgrid[0:9, 0:6].T.reshape(-1, 2)

# 存储对象点和图像点
objpoints = []  # 3D点
imgpoints = []  # 2D点

# 读取图像并检测角点
img = cv2.imread('calib.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
ret, corners = cv2.findChessboardCorners(gray, (9, 6), None)

if ret:
    objpoints.append(objp)
    corners2 = cv2.cornerSubPix(gray, corners, (11, 11), (-1, -1), criteria)
    imgpoints.append(corners2)

# 标定
ret, mtx, dist, rvecs, tvecs = cv2.calibrateCamera(
    objpoints, imgpoints, gray.shape[::-1], None, None
)
```

### 4. 双目相机标定

[双目相机标定](https://docs.opencv.org/4.x/d9/d0c/group__calib3d.html)需要额外计算：

- **旋转矩阵** $R$：右相机相对于左相机的旋转
- **平移向量** $T$：右相机相对于左相机的平移
- **本征矩阵** $E$：$E = [T]_\times R$
- **基础矩阵** $F$：$F = K^{-T} E K^{-1}$

**OpenCV双目标定**：
```python
# 分别标定两个相机
ret1, mtx1, dist1, rvecs1, tvecs1 = cv2.calibrateCamera(...)

# 双目标定
retval, cameraMatrix1, distCoeffs1, cameraMatrix2, distCoeffs2, \
R, T, E, F = cv2.stereoCalibrate(
    objpoints, imgpoints1, imgpoints2,
    mtx1, dist1, mtx2, dist2, image_size

```

使用标定结果对图像进行[畸变校正](https://docs.opencv.org/4.x/d9/d0c/group__calib3d.html#ga320b4577f653a2b5b31711c7b8e59de0)：


### 6. 标定精度评估

**重投影误差**（Reprojection Error）是评估标定精度的核心指标：

$$
\text{Error} = \sqrt{\frac{1}{N} \sum_{i=1}^{N} \|m_i - \hat{m}_i\|^2}
$$

其中 $m_i$ 为检测到的角点，$\hat{m}_i$ 为重投影后的角点。

- **优秀**：< 0.3 像素
- **良好**：0.3 - 0.5 像素
- **一般**：0.5 - 1.0 像素
- **较差**：> 1.0 像素

**学术贡献**：
- 1999年提出张正友标定法，发表于IEEE TPAMI
- 该方法利用平面标定板，只需拍摄多张不同角度的图像即可完成标定
- 相比传统的三维标定物方法，大大降低了标定成本和复杂度

**张正友标定法的优势**：
1. 标定板制作简单（普通打印机即可）
2. 不需要精确知道标定板的三维位置
3. 标定精度高，鲁棒性好
4. 被OpenCV、MATLAB等主流工具广泛实现

### Roger Tsai

Roger Tsai在1987年提出了经典的Tsai标定法，首次系统地将镜头畸变引入相机标定模型，为现代相机标定奠定了基础。
